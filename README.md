# AI Meal Planner & Calorie Scanner

A Flutter mobile app that generates personalized meal plans and estimates calorie 
content directly from a photo of your food, using on-device image recognition.

## Features

- 📸 Camera-based food recognition to automatically estimate calorie content
- 🍽️ Personalized meal plan generation
- 💾 Fully offline — all data stored locally, no account or server required

## Tech Stack

- **Framework:** Flutter
- **Language:** Dart
- **Image Recognition:** ML-based food detection
- **Storage:** SQLite (local, on-device)
- **IDE:** Android Studio

## Why I built this

I wanted to explore how machine learning could be applied to a practical, everyday 
problem — tracking nutrition without the friction of manual logging. This project 
combines mobile development with applied AI, two areas I'm actively building skills in.
